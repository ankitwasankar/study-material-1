Search instructions

Enter the word you are searching for. If you are using Navigator 4.0 or higher, or Internet Explorer 4.0 or higher, then the results will appear automatically. After you've entered the first three characters a list of pages about that subject will be shown. As you enter more characters this list will be updated while you enter the word you are searching for.

Because of the way this saerch works, you can only serach for one word at a time. But because the results are shown immediately, you can quickly change the search if the results aren't useful, or no match is found.

If the results do not appear automatically, then use the Go button to execute the search.

To restore the search results, after you've clicked on one of the results to view the corresponding page, use the Go button.

To show the selection list with all the pages again, use the same button that activated the search. While you are using the search facility this button changes into select.

NOTE
This search engine is completely written in JavaScript, so it will only work if you are using a browser that supports JavaScript.
