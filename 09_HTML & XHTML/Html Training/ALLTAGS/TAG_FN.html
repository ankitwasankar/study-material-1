<FN>...</FN>

Description

The WebTV interface implements the <fn> element to handle footnotes like anchor fragments defined with <a name="anchor name">. That is, clicking on a footnote reference simply scrolls to the definition.

Attributes


HREF
Use the href attribute to set the selectable link to the footnote using the id attribute.




ID
Description of the ID attribute
