Download instructions

Because I got requests to make these pages available in one file I zipped all the files and put them on the server for you to download. Before you download this file please read the legal notes.

Installation

To install the download version follow these steps:
  1. Download the site as a zip file, htl.zip (537 Kb), or as a Unix tar file, htl.tar.gz.
  2. Create a directory.
  3. Unzip file you downloaded in the directory you just created. Use an unzipper that can handle long filenames, for example WinZip. If your operating system does not support long filenames then the links on the pages will not work.
  4. Start your browser and load the file index.html in the directory you created.
  5. Tada!

Official mirrors

If you want to create a mirror for this site you can use the same zip file. Just create a directory on your server, put the zip file there and unzip the files.

To be listed on the homepage as an official mirror site you must conform to the following rules:

  • You must put all the files, including the htl.zip file on your server, without modifications, except for the adapted link to your own page.
  • You must supply me with an email address to be able to contact you.
  • You must supply me with the geographical position of the server where the mirror site actually resides.
  • You must keep the site updated when newer versions are released. I will inform you by email when an update is available.
  • The mirror site must not be presented inside a frame, but in the complete browser window. The current version checks for this, and corrects it when needed. See below how you can modify the site to present a way to link to your own site.
If you cannot conform to these rules, but do put the site online, I would appreciate it if you let know about it. That way I still can keep you informed of updates.

Advertise

To everyone who mirrors my site I offer the possibility to personalize your local copy. You can put your own icon, with a size of 100x50 pixels on the homepage of the site plus make it link to any address you want. This is how you do that:
  1. Open the file start.html, and look for the line containing the comment "Advertise your site :".
  2. Change the value of the href attribute of the link into the address you want to link to.
  3. Put an image in the directory of the site, with the name name.gif. This image must be 100 pixels wide and 50 pixels high.
