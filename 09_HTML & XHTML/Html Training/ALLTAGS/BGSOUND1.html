BGSOUND example

If your browser supports the <BGSOUND> element you should hear a piece of music.

This is specified using the following code :

<HTML>
<HEAD>
<BGSOUND src="bgsound.mid">
</HEAD>
<BODY>
BODY-elements
</BODY></HTML>
